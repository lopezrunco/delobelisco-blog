## Getting Started

### Prerequisites

* node.js
* npm

### Installation

1. Clone the repo

2. Install NPM packages
   ```sh
   npm i
   ```
3. Run app
    ```sh
    npm start
    ```
4. Build & deploy
    ```sh
    npm run deploy
    ```
